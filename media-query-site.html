Media Queries

Media queries allow us to make our sites responsive. We can use them to apply css styling only when certain conditions apply. For instance, our sites can look different depending on the size of the screen, or viewport a user is viewing our content with. we can also change the way our site appears if a user is printing out website, or using a a screen reader.
